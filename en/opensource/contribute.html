Contribute to Xefili.dev

Github

You can help improving the xefili.dev repos and websites by contributing on Github. If you found a minor issue, you can open a pull request or create an issue detailing the problem. Should there be a bigger problem with a page, open an issue describing the issue. Remember that some issues may already be fixed in a different branch.

Report problems

At the top of every documentation page is a link to report an issue. Please fill out the form, and I will check and fix the problem.
