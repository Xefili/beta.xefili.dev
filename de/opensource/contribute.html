Zu xefili.dev beitragen

Github

Du kannst über Github zu xefili.dev beitragen. Solltest du einen kleinen Fehler gefunden haben, erstelle eine Pull-Request oder öffne ein Issue auf Github. Sollte das Problem größer sein, erstelle ein Issue und beschreibe das Problem genauer. Denk daran, dass manche Probleme schon in anderen Branches behoben sein können!
